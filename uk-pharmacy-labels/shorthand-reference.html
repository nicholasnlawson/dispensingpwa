Dosage Quantities - Tablets

Tablet quantities follow these patterns:

  • nt - Where n is any number (e.g., 1t, 2t, 0.5t, 1.5t, 7t)
  • n-mt - Where n and m are numbers for a range (e.g., 1-2t for "Take ONE to TWO tablets")

Dosage Quantities - Capsules

Capsule quantities follow these patterns:

  • nc - Where n is any number (e.g., 1c, 2c, 3c)
  • n-mc - Where n and m are numbers for a range (e.g., 1-2c for "Take ONE to TWO capsules")

Dosage Quantities - Liquid Preparations

Liquid quantities follow these patterns:

  • nml - Where n is any number (e.g., 5ml, 10ml, 15ml)
  • n-mml - Where n and m are numbers for a range (e.g., 5-10ml for "Take 5-10ml")

Dosage Quantities - Inhalers

Inhaler puff quantities follow these patterns:

  • np - Where n is any number (e.g., 1p, 2p, 4p)
  • n-mp - Where n and m are numbers for a range (e.g., 1-2p for "Inhale ONE to TWO puffs")

Dosage Quantities - Drops

Drop quantities follow these patterns:

  • nd - Where n is any number (e.g., 1d, 2d, 3d)
  • n-md - Where n and m are numbers for a range (e.g., 1-2d for "Apply ONE to TWO drops")

Handwritten Information Placeholders

Special 'X' shorthand codes - These codes leave space for handwritten information on printed labels:

  • xt - Expands to "Take .......... tablets" (allows handwriting quantity)
  • xc - Expands to "Take .......... capsules" (allows handwriting quantity)
  • xp - Expands to "Inhale .......... puffs" (allows handwriting quantity)
  • xpa - Expands to "Apply .......... patches" (allows handwriting quantity)
  • xd - Expands to "Apply .......... drops" (allows handwriting quantity)
  • xml - Expands to "Take .......... ml" (allows handwriting quantity)
  • xDS - Expands to ".......... times a day" (allows handwriting frequency)

Duration Specifications

Duration codes follow these patterns:

  • n/7 - For any number of days (e.g., 15/7 for fifteen days)
  • n/52 - For any number of weeks (e.g., 3/52 for three weeks)
  • n/12 - For any number of months (e.g., 4/12 for four months)
